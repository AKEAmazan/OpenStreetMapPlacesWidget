# OpenStreetMapWidget

OpenStreetMapWidget allows you to add location picker and place autocompletion on your Flutter projet (Android and IOS).

## Getting Started

### Step 1

Add dependencies

   http: ^0.12.0+1
   diacritic: ^0.1.0
   flutter_map: ^0.9.0
   rxdart: ^0.23.1
   location: ^3.0.2
   progress_dialog: ^1.2.2
   json_annotation: ^3.0.0
